Add custom links to the Jellyfin Web side navigation menu. Links appear below Home on the regular Jellyfin home page (not in the admin Dashboard). After saving, hard-refresh that page (Ctrl+Shift+R).

Icons use Material Design Icons. Leave the icon blank to use the default "link" icon.
